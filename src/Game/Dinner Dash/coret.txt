STATE AWAL
- saldo 0
- total pesanan 0
- ada 3 pesanan
// yang ditampilin
- QUEUE pesanan = (no urut, durasi, ketahanan, harga)
- QUEUE masakan = (no urut, durasi)
- QUEUE sajian = (no urut, ketahanan)

- Jika jumlah pesanan <= 7 atau total pesanan != 15
    - meminta masukkan command

COOK <n>
- jika <n> ada di pesanan

    - semua durasi di masakan dikurang 1
    - jika ada durasi masakan <m> yang bernilai 0 :
        - hapus <m> dari masakan
        - tambah <m> ke sajian

    - semua ketahanan di sajian dikurang 1
    - jika ada ketahanan sajian <s> yang bernilai 0 :
        - hapus <s> dari sajian
        - set durasi <s> menjadi durasi awal
        - tambah <s> ke masakan
        
    - tambah <n> ke masakan

    - tambah 1 pesanan

- jika <n> tidak ada di pesanan
    - tampilkan pesan peringatan
    - minta masukkan command ulang

SERVE <n>
- jika <n> ada di sajian
    - hapus <n> dari sajian
    - saldo ditambahkan sesuai dengan harga <n>
    - pelanggan bertambah 1
    - beri pesan makanan berhasil disajikan

    - semua durasi di masakan dikurang 1
    - jika ada durasi masakan <m> yang bernilai 0 :
        - hapus <m> dari sajian, pesanan
        - beri pesan bahwa <m> sudah matang

    - semua ketahanan di sajian dikurang 1
    - jika ada ketahanan sajian <s> yang bernilai 0 :
        - hapus <s> dari sajian
        - hapus <s> dari pesanan
        - set durasi <s> menjadi durasi awal
        - tambah <s> ke masakan
        - beri pesan bahwa <s> sudah hangus dan dimasak ulang

    - tambah 1 pesanan

- jika <n> tidak ada di sajian
    - tampilkan pesan peringatan
    - minta masukkan command ulang
